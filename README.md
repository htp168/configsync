#### 使用说明

1.  同步路径：C:\Program Files\PLSQL Developer 13\Configsync

2.  配置说明：

3.  执行操作：
    ```git

    git clone -b plsql git@gitee.com:htp168/configsync.git Configsync
    
    // git push
    cd Configsync
    git push -u origin plsql
    
    // git pull
    cd Configsync
    git pull -f origin plsql
    
    ```
